# Turtle Tracker Android App

一个用于记录宠物龟体重、状态、喂食、换水、冬眠等日常信息的纯净应用。

## 当前功能
- 龟档案管理，包括：体重记录（配有体重趋势图）、喂食记录、换水记录（支持盐度登记）、状态记录、冬眠记录
- 冬眠记录，包括：入眠记录、冬化记录、冬眠检视、出眠记录、开食记录
- 以上记录信息支持查看历史记录，支持按关键字检索
- 待办提醒，包括：新龟的喂食、换水、体重登记提醒；所有龟的体重异常提醒、长期未换水提醒、龟状态异常持续提醒
- 数据的备份与恢复

## 技术方案
- React Native + Expo（prebuild 后本地 Android 构建）
- 本地存储：AsyncStorage
- 图表：react-native-chart-kit

## 计划更新
- 增加龟的医疗记录
- 增加一个知识库，包括常用的养龟知识、常见养龟误区，常见龟病介绍及处理方法
- 增加拍照和图库功能

## 不推荐使用的人群
- 持有保育/保护动物的，当你被突击上门时，手机会立刻被控制
